# Version 1 Log

Date: 2026-04-05

## Snapshot

This log records Version 1 of the Singapore Architecture Guide in its redesigned multi-page format.

## Included In Version 1

- Hero-first homepage with the headline `Explore Singaporean Architecture`
- Real Singapore overview map with category-colored project pins
- Local project media stored under `assets/`
- Project cards with hero-photo backgrounds and rating details revealed on hover
- Standalone project sub-pages via `project.html?project=<id>`
- Embedded video highlights for key projects
- Categories covering Residential, Cultural, Institutional, Hospitality, Civil Infrastructure, and Others
- Five rating criteria: Design Expression, Technical Execution, User Experience, Sustainability, and Context Agency
