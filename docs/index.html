How Recommendation Works

Each project is read through five criteria that balance expression with lived performance.

Design Expression

How legible, original, and spatially memorable the formal idea feels on site.

Technical Execution

How convincingly the detailing, envelope, structure, and climatic resolution support the design idea.

User Experience

How well circulation, comfort, orientation, and public experience hold up in use.

Sustainability

How convincingly the project handles climate, planting, reuse, and long-term environmental performance.

Context Agency

How much the project improves its urban setting instead of remaining a sealed object.
