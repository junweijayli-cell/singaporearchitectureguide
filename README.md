# Singapore Architecture Guide

This package is prepared for the `singaporearchitectureguide` GitHub repository and GitHub Pages publishing.

## Included

- `index.html` homepage
- `project.html` project detail page
- `guide.css`
- `shared.js`
- `main.js`
- `project.js`
- `data/app-data.js`
- `assets/` local images, plans, and map
- `.github/workflows/deploy-pages.yml` GitHub Pages workflow
- `.nojekyll`

## Publish Target

Upload the contents of this package to the root of the `singaporearchitectureguide` repository.

The included workflow deploys the site through GitHub Actions when changes are pushed to `main`.
