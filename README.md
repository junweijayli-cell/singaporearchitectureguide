# Singapore Architecture Guide

A static, filterable Singapore architecture guide with a map-first layout, category filters, illustrated project cards, and plan-style detail views.

## Files

- `index.html`: page structure
- `styles.css`: visual design and responsive layout
- `script.js`: project dataset, filtering, sorting, and detail rendering

## Open locally

Open `index.html` directly in a browser, or serve the folder with a simple static server such as:

```powershell
python -m http.server 8000
```
